
## 1 — Definir o eixo conceitual permanente da marca

Hidden Layer já tem um eixo forte:

transformação invisível entre entrada e saída.

Isso vira o núcleo semântico fixo.
Tudo que existir na marca deve caber dentro de pelo menos um destes três domínios:

✅ Estados

Ex:
Latent State
Hidden State
Intermediate State
Converged State

✅ Processos

Ex:
Signal Path
Gradient Flow
Iteration
Convergence

✅ Relações

Ex:
Input → Output
Mapping
Transformation
Projection

Isso evita que a marca se disperse.

## 2 — Criar uma gramática visual (isso é o diferencial)

Você já começou sem perceber.
As três estampas possuem:
formas geométricas simples
repetição
direção implícita
ausência de ícones óbvios
Agora formalize isso como regra:
Regras visuais Hidden Layer
apenas formas básicas (linhas, círculos, blocos)
sem ilustrações figurativas
sem símbolos tecnológicos literais
uso de espaço negativo
leitura limpa em 2 segundos

Isso permite criar infinitas peças mantendo identidade.

## 3 — Estrutura evolutiva das coleções

Em vez de “coleção verão/inverno”, use versões conceituais:

#### V1 — Fundamentals
(INPUT → HIDDEN → OUTPUT)
(LATENT STATE)
(SIGNAL PATH)

Base da marca.

#### V2 — Dynamics
Flow, convergence, feedback loops.

#### V3 — Abstractions
Formas mais minimalistas, quase criptográficas.

Quem compra cedo sente que está acompanhando a evolução.

## 4 — O ponto estratégico mais importante agora

Enquanto espera as camisas:

Crie um documento simples (1 página) chamado:

Hidden Layer — Design System v1

Com:
Ideia central (1 parágrafo)
Regras visuais
Tipografia
Espessuras padrão de linha

Área padrão da estampa
Posição da assinatura nas costas
Isso evita que a marca mude sem perceber ao longo do tempo.
