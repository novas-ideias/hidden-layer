# Hidden Layer — Design System v1

## 1. Ideia Central

Hidden Layer representa o que acontece entre entrada e saída — a camada invisível de transformação. A marca explora estados intermediários, processos internos e mudanças que normalmente não são observadas diretamente.

O objetivo não é representar tecnologia de forma literal, mas traduzir conceitos abstratos em formas simples, silenciosas e atemporais.

A marca existe no espaço entre processo e resultado.

---

## 2. Princípios Fundamentais

* Minimalista e técnico
* Discreto e não agressivo
* Conceito antes do logo
* Atemporal acima de tendências
* Sistema visual consistente e repetível

Toda nova estampa deve respeitar esses princípios.

---

## 3. Regra de Hierarquia

### Frente

A frente da camiseta apresenta o **conceito**.

### Costas

A parte traseira funciona como **assinatura** da marca.

O conceito sempre vem antes da identificação.

---

## 4. Estrutura Conceitual da Marca

Todas as estampas devem pertencer a um dos três domínios abaixo.

### 4.1 STATE — Estados

Representa condição, latência ou potencial.

Exemplos:

* LATENT STATE
* HIDDEN STATE
* STABLE STATE
* UNOBSERVED STATE

Características visuais:

* formas estáticas
* equilíbrio
* redes ou estruturas estáveis

---

### 4.2 FLOW — Fluxo

Representa movimento, direção e propagação.

Exemplos:

* SIGNAL PATH
* FLOW FIELD
* CONVERGENCE
* DIVERGENCE
* PROPAGATION

Características visuais:

* linhas direcionais
* continuidade
* sensação de movimento implícito

---

### 4.3 TRANSFORM — Transformação

Representa mudança entre estados ou representações.

Exemplos:

* INPUT → HIDDEN → OUTPUT
* GRADIENT
* MAPPING
* PROJECTION
* THRESHOLD

Características visuais:

* transições
* deformações
* mudanças de padrão

---

## 5. Regras Visuais

* Utilizar apenas formas geométricas simples
* Evitar ícones ou símbolos tecnológicos literais
* Priorizar espaço negativo
* Leitura clara a distância
* Baixo ruído visual

A estampa deve funcionar sem explicação textual.

---

## 6. Linguagem da Marca

* Inglês para conceitos e nomes das estampas
* Português apenas para operação da loja
* Textos curtos e descritivos
* Evitar slogans e linguagem promocional

A marca deve parecer descoberta, não vendida.

---

## 7. O Que NÃO é Hidden Layer

* Estética sci‑fi ou futurista explícita
* Elementos tecnológicos óbvios (chips, circuitos, robôs)
* Ilustrações figurativas
* Tendências visuais passageiras
* Excesso de detalhe ou complexidade

Se a estampa precisar de explicação longa, ela não pertence ao sistema.

---

## 8. Evolução da Marca

A Hidden Layer evolui por versões conceituais, não por coleções sazonais.

### V1 — Fundamentals

* INPUT → HIDDEN → OUTPUT
* LATENT STATE
* SIGNAL PATH

### V2 — Dynamics (futuro)

* CONVERGENCE
* GRADIENT
* THRESHOLD

### V3 — Abstractions (futuro)

* MAPPING
* PROJECTION
* SYSTEM

Cada nova versão deve expandir o sistema, nunca substituí-lo.

---

## 9. Regra Final

Antes de criar qualquer nova estampa, responder:

1. Funciona sem explicação?
2. Continua minimalista a 2–3 metros?
3. Parece atemporal?

Se qualquer resposta for não, a estampa deve ser descartada.
